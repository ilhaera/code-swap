Code-swap
==========
Easily swap code pieces


Installation
------------
Open **Settings** â†’ **Install** and search for `Code-swap`.

Or install through command-line:

	apm install code-swap

Keymap
------

do it

  ```ctrl-alt-q``` to generate code-swap template
  ```
  ////code1
                    activated
  /*///code2
                    comment
  /* *///"
  ```
```ctrl-alt-s``` to swap code pieces
```
/*///code1
                    comment
/*///code2
                    activated
/* *///"
```
